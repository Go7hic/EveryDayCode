#js# Js 单体模式

dyygtfx Updated at Feb 11, 2015

第二次创建的对象与第一次完全相同，预期的结果如下 var a = new Test() var b = new Test() a === b; //结果为 true

实现方法： 1.通过静态属性

function Test() {
    if (typeof Test.instance === "object") {
        return Test.instance;
    }

    this.start_img = 0;
    this.bang = "Big"

    Test.instance = this;//缓存

    // 隐式返回
    return this;
}
var a = new Test();
var b = new Test();
a === b

2.通过闭包

function Test() {
    var instance = this;
    this.start_img = 0;
    this.bang = "Big"

    Test = function () {
        return instance;
    }
}
var a = new Test();
var b = new Test();
a === b
