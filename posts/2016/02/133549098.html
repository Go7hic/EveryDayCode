编写一个方法 求一个字符串的字节长度

dyygtfx Updated at Feb 14, 2016

 function GetBytes(str){

        var len = str.length;

        var bytes = len;

        for(var i=0; i<len; i++){

            if (str.charCodeAt(i) > 255) bytes++;

        }

        return bytes;

    }
alert(GetBytes('hello,你好'))
